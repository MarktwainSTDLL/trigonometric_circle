# Quadrantes

Explicação básica a respeito dos quadrantes do Ciclo Trigonométrico.

Primeiro quadrante é composto por:
  ```
     -> 30°
     -> 45°
     -> 60°
  ```
Segundo quadrante é composto por:
  ```
     -> 120°
     -> 135°
     -> 150°
  ```
Terceiro quadrante é composto por:
```
     -> 210°
     -> 225°
     -> 240°
```
Quarto quadrante é composto por:
```
     -> 300°
     -> 315°
     -> 330°
```
